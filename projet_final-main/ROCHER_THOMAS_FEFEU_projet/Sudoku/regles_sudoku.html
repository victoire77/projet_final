Les règles du Sudoku

Avant de jouer voici un récapitulatif des règles du jeu pierre feuille ciseaux. Prenez le temps qu'il vous faudra pour les lire.
(Si vous considérez ne pas avoir besoin d'explications ou lorsque vous aurez fini de les lire vous pourrez cliquer sur "Skip" pour accéder au jeu.)


Quel est le but du jeu ?

Le but est de compléter chaque case de la grille avec un chiffre allant de 1 à 9, en respectant simultanément 3 contraintes:

  1. Le chiffre ne doit pas déjà se trouver sur la même colonne.
  2. Le chiffre ne doit pas déjà se trouver sur la même ligne.
  3. Le chiffre ne doit pas déjà se trouver dans le même bloc.


Comment jouer ?

Pour jouer, vous devez cliquer sur la case dont vous pensez avoir trouver le chiffre. Vous cliquez ensuite sur le chiffre que vous pensez avoir deviné.
S'il s'agit du bon, alors le point d'interrogation sera remplacé par le chiffre choisi. Dans le cas d'une erreur, vous perdez une vie. Vous avez au total 5 vies.
Si vous les perdez toutes, vous perdez le jeu.
